# ERIR

Для запуска:
* Установить [NodeJS](https://nodejs.org/en/download/)
* Запустить NodeJS Command prompt(Windows)

>(перейти в папку ERIR)

```
cd .../ERIR 
```
```
npm install
```
```
npm start
```
* [http://localhost:8080](http://localhost:8080)

_Все изменения в коде сразу подгружаются в браузере_
